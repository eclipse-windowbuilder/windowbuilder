= RCP PropertyPage Wizard

image:images/wizard_rcp_propertypage0.gif[image]

Subclasses of the Eclipse RCP *PropertyPage* class can be created using the *RCP PropertyPage* wizard. The wizard can be
selected from the drop down Designer wizard menu or from the Eclipse *New* wizard.

To use the wizard, select the project source folder and package to contain the class. Then enter the class name and hit
the *Finish* button.

--
image:images/wizard_rcp_propertypage1.gif[image]
image:images/wizard_rcp_propertypage2.gif[image]
--

The wizard generates the following code.

[source,java]
----
import org.eclipse.swt.SWT;
import org.eclipse.swt.widgets.Composite;
import org.eclipse.swt.widgets.Control;
import org.eclipse.ui.dialogs.PropertyPage;
 
public class RcpPropertyPage extends PropertyPage {
	public RcpPropertyPage() {
		super();
	}
	@Override
	public Control createContents(Composite parent) {
		Composite container = new Composite(parent, SWT.NULL);
		return container;
	}
}
----

Property Pages can use any widget and layout manager.
