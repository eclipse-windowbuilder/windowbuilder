= Swing Absolute Layout

*Absolute Layout* or *Null Layout* is a simple x,y oriented layout. During layout, the live component is shown moving
with a tooltip showing its current location or size. Dynamic snap / alignment points are provided to make it convenient
to align the component with other components or the window margin. Resizing a component also snaps to the preferred size
of the resized component or to the size (vertical or horizontal) of any other component on the same parent.
